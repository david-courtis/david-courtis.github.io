Personal fork of [LibreChat](https://librechat.ai/) that extends the upstream multi-provider chat UI with:

- **Standalone image generation** — a NovelAI-inspired interface kept separate from the chat flow, so image generation is a first-class capability rather than a chat sidecar.
- **Additional endpoint support** — extra OpenAI-compatible providers and configuration options layered on top of upstream's already-broad provider list (Anthropic, AWS Bedrock, OpenAI, Azure OpenAI, Google, Vertex AI, custom).
